# 工业感极简白底控制台前端风格规范

本规范用于当前仓库所有后续前端页面与组件开发，默认必须遵循。

## 1. 设计定位

- 风格关键词：工业感、极简、理性、克制、工程化。
- 主体基调：白底、黑灰文本、细边框、轻阴影。
- 信息表达优先级：清晰度 > 装饰性。

## 2. Design Tokens（必须使用 CSS 变量）

| Token | Value | 用途 |
| --- | --- | --- |
| `--bg` | `#FFFFFF` | 页面主背景 |
| `--text` | `#000000` | 主文本 |
| `--muted` | `#666666` | 次级文本/注释 |
| `--line` | `#EAEAEA` | 常规分割线/边框 |
| `--line-2` | `#E0E0E0` | 强调级边框 |
| `--panel` | `#FAFAFA` | 面板背景 |
| `--accent` | `#FF5722` | 操作强调色 |
| `--success` | `#4CAF50` | 成功/在线状态 |

推荐结构 token：

- `--radius-sm: 4px`
- `--radius-md: 6px`
- `--radius-lg: 8px`
- `--shadow-soft: 0 2px 8px rgba(0, 0, 0, 0.06)`

## 3. 字体系统

- 正文字体：`Space Grotesk` + `Noto Sans SC`。
- 数字/标签/技术信息：`JetBrains Mono`。
- 禁止使用可爱化、圆润娱乐化字体。

## 4. 组件视觉规则

- Header：固定 `60px` 高度，底部 `1px` 边框。
- 分段切换器：扁平胶囊，圆角 `6px`，细边框，激活态仅轻微底色差。
- 按钮：细边框，Hover 背景 `#fff -> #f5f5f5`，过渡 `0.2s`。
- 卡片/面板：圆角 `4~8px`，极轻阴影（使用 `--shadow-soft`）。
- 大区域背景：可使用浅点阵网格（`radial-gradient`）增强控制台感。

## 5. 动效规则（只允许微动效）

- 允许：状态点 `pulse`、刷新图标 `spin`、hover `0.2s` 过渡。
- 禁止：夸张动画、长时复杂动效、跳跃式过渡。

## 6. 布局规则

- 基础结构：`Header + 主工作区（Toolbar + Panels）`。
- 面板区至少包含：状态条、数据卡片、日志区或拓扑区（可用 mock 数据）。
- 响应式要求：桌面 + 移动端可用；关键断点 `768px`。

## 7. 技术实现约束（本仓库默认）

- 默认栈：`Vite + Vue3 + SFC`。
- 样式要求：使用全局 CSS Variables；避免内联硬编码色值。
- 组件结构：语义化清晰，命名工程化，不使用花哨装饰命名。

## 8. 反向约束（禁止项）

- 禁止暗色主题（除非明确新增单独主题任务）。
- 禁止霓虹紫/赛博蓝主视觉。
- 禁止大面积毛玻璃效果。
- 禁止超大圆角和“卡哇伊”视觉风格。

## 9. 交付前自检清单

- 是否使用了规定色彩 token？
- 是否保持白底 + 黑灰文字 + 细边框？
- 是否只使用微动效？
- 是否在 `768px` 断点下保持可用？
- 是否触发了任何禁止项？
